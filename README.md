# RoomAnnotationProcessor
This repository demonstrate usage of multi-module android project. The repository has three modules.
     
   1. `annotation` - This module is used to define annotation definition.
   
   2. `annotation-processor` - This module is used to process the classes annotated with user defined annotation.
   
   3. `app` - This is a sample android application demonstrating usage of annotation and annotation processor.
